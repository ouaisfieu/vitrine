CLONER CE MANANT

Pourquoi et comment adapter cette vitrine

Ce projet peut être utilisé ou adapté par votre groupe, votre collectif ou toute personne souhaitant transmettre un message de manière simple, directe et visuelle.

Ce que permet cette vitrine

  • Diffuser une vidéo hébergée en ligne sans dépendre de plateformes commerciales.

  • Afficher des messages ou slogans sous forme de sous-titres animés.

  • Proposer des actions concrètes via des boutons visibles.

  • Créer une porte d’entrée vers un univers narratif, une initiative locale ou une action collective.

Pour qui ?

  • Un collectif souhaitant marquer les esprits avec un message fort.

  • Une association qui veut proposer un point d’entrée original à ses campagnes.

  • Un groupe informel qui cherche à rendre visible une cause ou un état d’esprit.

  • Un·e créateur·rice indépendant·e qui souhaite mettre en scène une ambiance, un ton, une tension.

Comment s’en servir

  1. Télécharger ou copier les fichiers

    • Aucun logiciel particulier n’est nécessaire.

    • Un simple éditeur de texte permet de modifier le contenu.

  2. Remplacer la vidéo

    • Hébergez une vidéo de votre choix, dans un format .mp4.

    • Mettez à jour le lien dans le fichier index.html.

  3. Adapter les sous-titres

    • Le fichier subtitles.vtt peut contenir des slogans, des phrases fortes, ou même des extraits sonores mis en texte.

    • Le rythme de défilement peut être ajusté.

  4. Modifier les boutons

    • Les actions peuvent rediriger vers une page, un formulaire, une ressource ou tout autre contenu en ligne.

  5. Publier le tout

    • GitHub Pages, un simple hébergement mutualisé ou même un serveur local suffisent pour mettre la vitrine en ligne.

Et après ?

Ce point de départ peut être enrichi. Il est possible de :

  • Ajouter un formulaire de contact (ou non).

  • Intégrer un appel à action.

  • Créer des suites scénarisées (cliquer sur un bouton ouvre une nouvelle étape).

  • Monter une version collective, en réseau, où chaque groupe adapte la sienne.

Cette vitrine est conçue pour rester simple, duplicable et libre.
Elle peut être utilisée comme outil de communication, de mobilisation ou de narration.
